# Веб-приложение CRM-системы транспортной компании

Веб-приложение для автоматизации бизнес-процессов транспортной компании. Система оптимизирует управление персоналом и автопарком, автоматизирует взаимодействие с клиентами, контролирует жизненный цикл заказов и предоставляет инструменты для глубокой аналитики и мониторинга ключевых показателей эффективности (KPI).

---

## Режимы работы приложения

Приложение разделено на два ключевых функциональных режима:

### 1. Администрирование
Предназначен для управления ресурсами компании.

* **1.1. Управление сотрудниками:**
    * Полный цикл CRUD (создание, чтение, обновление, удаление/архивация).
    * *Состав сущности:* ФИО, Электронная почта (`email`), Контактный телефон, Адрес проживания.
    * *Валидация:* Проверка email, телефонных номеров, проверка на уникальность критических данных.
* **1.2. Управление автопарком:**
    * Полный цикл CRUD для транспортных средств (ТС).
    * *Состав сущности:* Название/Модель, Дата регистрации/выпуска, Полезный объем кузова ($m^3$).
    * *Классификация ТС:*
        * **Тип кузова:** Бортовые, Тентованные, Фургоны, Рефрижераторы, Самосвалы.
        * **Категория:** Малотоннажные (`N1`), Среднетоннажные (`N2`), Крупнотоннажные (`N3`).
    * *Валидация:* Проверка корректности ввода дат, валидация числовых диапазонов объема и грузоподъемности.

### 2. Работа с клиентами и заказами
Обеспечивает работу менеджеров по логистике и работе с клиентами.

* **2.1. Регистрация клиентов:** Быстрый ввод клиентов в систему с созданием профиля.
* **2.2. Регистрация заказов:** Прием заявок на транспортировку, назначение исполнителей и ресурсов.
* **2.3. Отчетность:** Формирование документов за выбранные периоды времени.
